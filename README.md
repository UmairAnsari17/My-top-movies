# 🎬 My Top Movies

A Flask web application that allows users to search, save, rate, review, and manage their favorite movies.

## 🚀 Live Demo

https://my-top-movies-diri.onrender.com

## ✨ Features

- User Registration & Login
- Search movies
- Add movies to your collection
- Rate and review movies
- Update and delete movies
- Responsive UI
- Light/Dark Mode
- PostgreSQL Database
- Deployed on Render

## 🛠️ Technologies Used

- Python
- Flask
- SQLAlchemy
- Flask-Login
- PostgreSQL
- Bootstrap 5
- HTML/CSS
- JavaScript
- TMDb API

## ⚙️ Installation

```bash
git clone https://github.com/UmairAnsari17/My-top-movies.git

cd My-top-movies

pip install -r requirements.txt

python main.py
```

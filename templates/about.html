About My Top Movies

Organize, rate, and discover your favorite movies all in one place.

🎬 About the Project

My Top Movies is a Flask-based web application that allows movie enthusiasts
to search, add, rate, review, and organize their favorite movies.
The application uses The Movie Database (TMDb) API to fetch movie information,
including posters, descriptions, and release dates.

✨ Features

  • User Registration and Secure Login
  • Search Movies using TMDb API
  • Add Movies to Your Personal Collection
  • Rate Movies (0–10)
  • Write Personal Reviews
  • Movie Ranking Based on Ratings
  • Dark/Light Theme
  • Responsive Design

⚙️ How It Works

  1. Create an account.
  2. Login securely.
  3. Search for a movie.
  4. Select the movie from the search results.
  5. Rate and review it.
  6. Manage your personal movie collection.

⭐ Why My Top Movies?

Instead of keeping track of movies manually, My Top Movies provides a
simple and user-friendly interface where users can organize their favorite
movies, maintain ratings and reviews, and access accurate movie information
through the TMDb API.

🚀 Future Enhancements

  • Movie Watchlist
  • Favorite Movies
  • Recommendation System
  • User Profiles
  • Movie Genres Filter
  • Search History
  • Profile Picture Upload
  • Email Verification
